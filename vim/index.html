作为一款强大的编辑器，原版的vim居然没有自动补全、自动缩进等功能。因此呢，打造一款超越vscode，且属于自己的代码编辑器vim就必须安装些插件了。

Vim Awesome

该网站提供了很多vim插件供你选择

安装vimPlug

vimPlug是vim的一个智能插件管理工具，使用vimPlug可以让你几乎只要一条命令就能方便的安装一个vim插件。

在家目录中创建./.vim/autoload/

进入下载

curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim

安装插件

我们在家目录创建文件.vimrc

touch .vimrc

并键入：

call plug#begin('~/.vim/plugged')

call plug#end()

以后我们安装插件时，只需在这两句话之间 Plug ‘plug_name’，保存后在普通模式下用命令:PlugInstall即可安装所有插件。

自动补全插件的安装

本来我想使用YouYouCompleteMe的，但奈何这插件所需环境太难配置，遂弃之用了coc-nvim。

coc-nvim也是一款自动补全插件。

确认vim版本

在安装coc-nvim之前需要确认vim版本大于8.1。

需要升级时，先要卸载原先的vim版本

yum -y remove vim

在github上找到新版vim，可以找到vim9.0.0236并下载

wget -O vim9.0.0236.zip https://github.com/vim/vim/archive/v9.0.0236.zip

之后可用unzip解压安装包

unzip vim9.0.0236.zip

此时进入vim9.0.0236目录下的./src目录并执行以下命令安装

./configure --prefix=/usr/local/vim9/ --enable-pythoninterp=yes --with-python-config-dir=/usr/lib64/python2.7/config
make && make install

此时可能会出现no terminal library found错误

解决方法：

yum -y install ncurses-devel.x86_64

并重新执行以上命令

之后将vim命令链接到新安装的vim9上

ln -s /usr/local/vim9/bin/vim /usr/bin/vim

安装nodejs和yarn

由于该插件需要nodejs和yarn的支持，故需要安装二者

nodejs的安装略…

yarn的安装：

npm install -g yarn

cd ~/.vim/plugged/coc.nvim
yarn install

安装clangd

可在GitHub中下载clangd的zip包并解压至安装目录。

clangd下载地址如下：clangd/clangd: clangd language server (github.com)

wget https://github.com/clangd/clangd/releases/download/14.0.3/clangd-linux-14.0.3.zip

unzip clangd-linux-14.0.3.zip

之后只需找到解压后包里的clangd二进制文件并建立软链接

cd ./clangd-linux-14.0.3/bin
//其中包含文件clangd
ln -s /opt/clangd/bin /usr/local/bin/

此时键入clangdb命令若包含clangd的信息则说明安装成功

自动补全C/C++

有两种方案

  • 安装coc-clangd插件，在.vimrc文件命令模式下键入

    ：CocInstall coc-clangd

  • 在.vimrc文件命令模式下键入

    :CocConfig

    进入coc-settings.json文件中，输入

    {
    "languageserver": {
    "clangd": {
    "command": "clangd",
    "rootPatterns": ["compile_flags.txt", "compile_commands.json"],
    "filetypes": ["c", "cc", "cpp", "c++", "objc", "objcpp"]
    }
    }
}

保存并退出即可完成
